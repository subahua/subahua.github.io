2-1:Android手机环境

  • 安装Git,Node,SSH

2-2:Linux(Ubuntu)环境搭建

  • 安装Git，Nodejs

    2-3:Windows环境搭建

  • 安装Git，官网链接:https://git-scm.com/
  • 安装Node，官网链接(建议下载长期版):https://nodejs.org/en/

3：安装Hexo框架

  • 如果是Android手机,进入Termux,键入命令:npm install -g hexo-cli
  • 如果为Ubuntu,进入终端,键入命令:sudo npm install -g hexo-cli
  • Windowns环境，进入GitBash，键入命令:npm install -g hexo-cli
  • 提示以上的命令行在以下文章简称Termux
  • 如果npm命令网速慢，可以更换npm软件源,命令为:npm config set registry https://registry.npm.taobao.org

    4:Hexo框架的一些基础命令

5:创建网站(Blog)

  • 在Termux中键入命令:hexo init Blog
  • 进入刚刚生成的文件夹Blog,命令：cd Blog
  • 键入命令hexo s
  • 在浏览器中链接输入:http://localhost:4000/，即可。
  • 效果图展示 **好了👌，到目前位置，Blog已经搭建成功，就差部署到Github上了**
